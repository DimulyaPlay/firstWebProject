1. Сохранение зависимостей проекта

На исходном компьютере, где есть доступ к интернету, выполните следующие шаги:

    Активируйте виртуальное окружение вашего проекта.
    Сохраните список всех установленных в нем пакетов с их версиями в файл requirements.txt, используя команду:

cmd

pip freeze > requirements.txt

    Скачайте все зависимости из requirements.txt в папку для последующего переноса на целевой компьютер. Вы можете сделать это, используя команду pip download:

cmd

pip download -r requirements.txt -d ./deps/

Эта команда скачает все пакеты, перечисленные в requirements.txt, и сохранит их в папку deps/.
2. Копирование проекта и зависимостей

Перенесите ваш проект вместе с папкой deps/, содержащей скачанные зависимости, и файлом requirements.txt на целевой компьютер через USB-накопитель, сетевой диск или любой другой доступный способ.
3. Установка зависимостей на целевом компьютере

На целевом компьютере выполните следующие шаги:

    Создайте виртуальное окружение (если это еще не сделано) в директории вашего проекта:

cmd

python -m venv venv

    Активируйте виртуальное окружение. На Windows это делается командой:

cmd

venv\Scripts\activate

    Установите зависимости, используя пакеты из папки deps/. Убедитесь, что вы находитесь в директории проекта, где лежит папка deps, и активировано виртуальное окружение:

cmd

pip install --no-index --find-links=./deps -r requirements.txt

Эта команда установит все необходимые зависимости из локальной папки deps, не требуя подключения к интернету.
